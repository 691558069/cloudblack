# 后续开发计划

## AI 审核助手

目标：引入 AI 作为审核辅助，不直接自动通过云黑记录，降低误伤风险。

建议能力：

- 定时扫描待审核记录
- 分析 QQ、主体名称、关联账号、标签、原因、严重程度
- 输出审核建议：建议通过 / 建议拒绝 / 需要人工复核
- 输出置信度、风险理由、证据充分性判断
- 标记疑似重复主体或重复提交
- 后台审核列表展示 AI 建议和理由
- 管理员仍作为最终审核人

建议新增配置：

- AI 接口地址
- AI API Key
- 模型名称
- 是否启用 AI 审核助手
- 自动扫描间隔
- 最低置信度阈值

建议新增字段：

- `ai_status`
- `ai_suggestion`
- `ai_confidence`
- `ai_reason`
- `ai_checked_at`

建议接入方式：

- 优先兼容 OpenAI `/v1/chat/completions` 格式
- 可接 New API 网关
- 可接本地 Ollama 或其他兼容服务

注意：

- AI 默认只给建议，不自动写入正式云黑名单
- 若后续启用自动处理，也只建议用于明显灌水、证据为空、重复提交等低风险拒绝场景
- 高风险通过必须人工确认
